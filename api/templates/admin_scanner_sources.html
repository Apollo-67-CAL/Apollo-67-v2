How Controls Work

  • Blocking hides a source from the scanner Sources popup.
  • Display name lets you alias labels (for example, Twitter to X).
  • Min mentions and min confidence filter noisy sources.
  • Weight is stored for transparency and future tuning; it does not change signal or trade logic now.
