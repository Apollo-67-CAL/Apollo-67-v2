Trading Tactics Overview

A tactic defines a configurable trading style. Enabling a tactic makes it available to the admin configuration layer; activation chooses which profile is currently applied for interpretation.

How To Add A Tactic

  1. Click Create Tactic.
  2. Enter name, description, tags, and parameters JSON.
  3. Save the tactic, then enable/disable it from the table as needed.
